# Joe Sharp's Personal Site ✨

A minimalist Jekyll-powered project display.

## 🚀 Quick Start

```bash
# Install dependencies
bundle install

# Start the development server
bundle exec jekyll serve
```

## 🧙‍♂️ Features

- Clean, minimalist design
- Responsive layout
- Blog support
- Easy customization

## 🛠️ Tech Stack

- Jekyll
- SCSS
- JavaScript
- Font Awesome
